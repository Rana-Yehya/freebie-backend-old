# Stage One

Basic setup for a store with multiple products.

## Features

- Add products to the store
- Add products to the shopping cart
- Remove products from the shopping cart
- Calculate the total price of the shopping cart
- Nake orders, and track them

# Stage Two

Add subscription to stores.

## Features

- Add Basic, Sliver, Gold, Diamond subscriptions to the store
- Change the subscription of the store manually
- Automatically renew the subscription of the store every month
- When a store adds a Discount, it is automatically applied to the products
- When a store adds a Discount and one of the products already has a discount, it is automatically calculated and the highest discount is applied
- Ads have limited time can be edited by admins
- Adding boxes
